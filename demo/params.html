参数简介
参数 类型 默认值 是否必须 说明
croperWidth Number 800 是 整个裁剪画布宽度
croperHeight Number 600 是 整个裁剪画布宽度
width Number 300 是 最终裁剪生成图片宽度，即裁剪框宽度
height Number 300 是 最终裁剪生成图片高度，即裁剪框高度
shadowColor String(HEX、RGB) rgba(0,0,0,0.7) 否 画布背景色
maxSize Number 2000 否 被裁剪图片最大边长
lineColor String(HEX、RGB) #fff 否 裁剪框边线颜色
dashLineColor String(HEX、RGB) rgba(255,255,255,0.8) 否 裁剪框中间四条虚线颜色
edgeWidth Number 3 否 裁剪框四边短线宽度
imgType String image/png 否 裁剪结果图片类型，可用值：image/jpg、image/jpeg、image/png
quality Number 1 否 裁剪结果图片质量，imgType为image/jpg或image/jpeg有效
offBackgroundColor String(HEX、RGB) rgba(0,0,0,0.6) 否 未开始裁剪时，裁剪框周围阴影颜色
onBackgroundColor String(HEX、RGB) rgba(0,0,0,0.2) 否 开始裁剪时，裁剪框周围阴影颜色
scalable Boolean true 否 是否可缩放
scaleStep Number 0.02 否 每次缩放间距
created Function none 否 裁剪面板创建完成回调函数
beforeDrag Function none 否 开始移动裁剪框回调函数
dragged Function none 否 移动裁剪框结束回调函数
beforeScale Function none 否 开始缩放回调函数
scaled Function none 否 缩放结束回调函数
beforeReset Function none 否 开始重置回调函数
reseted Function none 否 重置结束回调函数
beforeDestroy Function none 否 开始销毁回调函数
destroyed Function none 否 完成销毁回调函数
属性简介（所有设置的参数均只读）
参数 类型 读写 说明
canvas HTMLElement 只读 整个画布元素
x Number 只读 被裁减图片x坐标（距离画布左侧距离）
y Number 只读 被裁减图片y坐标（距离画布上端距离）
scale Number 只读 当前图片缩放值（1为原图尺寸）
scaling Function 只读 设置裁剪图片缩放值：scaling(scale)
@params scale; Number;缩放值
reset Function 只读 重置裁剪图片状态，无参数则恢复默认状态：reset(x,y,scale)
@params x; Number;裁剪图片x值
@params y; Number;裁剪图片y值
@params scale; Number;裁剪图片缩放值值
cut Function 只读 裁剪图片，返回裁剪框大小图片：cut()
@return String(base64);裁剪图片base64字符串。
为保证图片清晰度，所有裁剪图片放大了，可按参数中设置的裁剪尺寸设置图片大小
destroy Function 只读 销毁裁剪画布，释放箭头事件：destroy()
